ShowStash (Prime Solo Project)

showstash.heroku.com

ShowStash is a web application that keeps a record of concerts a user has attended. The app replaces the need to track concerts in a spreadsheet or having a collection of ticket stubs to remember your concert history.
Tech Used: PostgreSQL, Express, AngularJS, Node.js, Bulma, Filestack, setlist.fm and last.fm RESTful web services, Angular Toastr.
